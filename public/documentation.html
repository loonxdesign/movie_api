MyFlix API Documentation

Intro

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Business Logic

URL

HTTP Method

Request body data format

Response body data format
Return a list of ALL movies /movies GET None A JSON object holding an array with all movies.
Return data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie containing description, genre, director, image URL, and whether it's featured or not properties.
Return data about a genre by name movies/genres/:genreName GET None A JSON object holding data about a genre containing name and description properties.
Return data about a director by director's name movies/directors/:directorName GET None A JSON object holding data about a director containing name, bio, birth year, and death year properties.
Allow new users to register /users POST A JSON object holding user data containing username, password, email, and birthday properties. A JSON object holding user data containing id, username, password, email, and birthday properties and an array for placing favourite movies.
Allow users to update their user info /users/:username PUT A JSON object holding the user info that should be changed. It should contain at least one of the following key/value pairs: Username/String, Password/String, Email/String, and/or Birthday/Numbers. A JSON object holding updated user data containing id, username, password, email, and birthday properties and an array for placing favourite movies.
Allow users to add a movie to their list of favourites /users/:username/movies/:title POST None A JSON object holding updated user data containing id, username, password, email, and birthday properties and an array for placing favourite movies.
Allow users to remove a movie from their list of favourites /users/:username/movies/:title DELETE None A JSON object holding updated user data containing id, username, password, email, and birthday properties and an array for placing favourite movies.
Allow existing users to deregister /users/:username DELETE None A text message indicating whether a user was successfully removed.
