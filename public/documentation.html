MyFlix API Documentation

Intro

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Get ALL movies /movies GET None A JSON object holding an array with all movies.
Get movie by title /movies/:Title GET None A JSON object holding data about a single movie containing ID, title, year, description, genres, director, and image URL.
Get movies by genre name /movies/genres/:genresName GET None JSON object showing data for all movies that share a genre type.
Get movies by director name /movies/director/:directorName GET None JSON object showing data for all movies that share a director.
Get data about a director by name /directors/:directorName GET None A JSON object holding data about a director containing name, bio, birth year, and death year properties.
